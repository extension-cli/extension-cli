# Security Policy

## Supported Versions

Only the latest published version is supported with security fixes.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Send a private report to: security@extension-cli.dev

Include:

1. Affected version and environment.
2. Reproduction steps or proof of concept.
3. Impact assessment.
4. Any suggested remediation.

We will acknowledge receipt within 3 business days and provide status updates as we triage and fix the issue.
