# Support

## Community Support

For usage questions, troubleshooting, and discussion:

1. Open a GitHub issue: https://github.com/extension-cli/extension-cli/issues

## Bug Reports

When filing issues, include:

1. Exact command and flags.
2. Output and error message.
3. Environment details (`node -v`, `pnpm -v`, OS, browser).
4. Steps to reproduce.
